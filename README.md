# booQ-UI

![main](https://github.com/traPtitech/booQ-UI/workflows/main/badge.svg)

booQ Frontend

## 開発環境構築

実行には Node.js が必要です

```shell
$ npm i
$ npm run gen-api
```

## コマンド

### ホットリロード環境

```shell
$ npm run dev
```

**実行前に一度`$ npm run gen-api`を実行する必要があります**

### ビルド

```shell
$ npm run build
```

ビルド

```shell
$ npm run gen-api
```

API クライアントコード生成
